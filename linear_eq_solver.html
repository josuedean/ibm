Simple Linear Equation Solver

Define an equation of the form z = C1x1 + C2x2 + ..., leave unknown values blank, and solve what is missing. You may leave z blank as well. If all variable values are known, the page will calculate z. If there is only one unknown variable, the page gives the direct answer. If there are multiple unknowns, the page can list integer solutions within the bounds you set.

Inputs

Output

Current equation
495000 = 450x1 + 550x2
Enter coefficients and optional known values, then press Solve. You can also leave z blank.
Notes: One equation with several unknowns usually does not have a single unique answer. When that happens, this tool either shows all bounded integer solutions or explains why the result is not unique.
